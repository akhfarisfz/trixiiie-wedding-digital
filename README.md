# Trixiiie Wedding Digital - Undangan Digital Premium

Selamat datang di repositori **Trixiiie Wedding Digital**, sebuah platform landing page untuk layanan pembuatan undangan pernikahan digital yang elegan, modern, dan responsif.

## ✦ Fitur Utama

- ⚡ **Pengerjaan Cepat:** Undangan selesai dalam waktu kurang dari 24 jam.
- ✏️ **Revisi Sepuasnya:** Bebas revisi sampai hari-H untuk memastikan kepuasan Anda.
- 👥 **Tamu Tak Terbatas:** Kirim undangan ke sebanyak mungkin tamu tanpa biaya tambahan.
- 🎵 **Custom Musik:** Bebas pilih lagu latar favorit untuk undangan Anda.
- 💎 **Desain Premium:** Pilihan tema yang estetik dan terlihat profesional di semua perangkat.

## 🎨 Pilihan Tema

Proyek ini menampilkan katalog tema undangan digital yang siap dipesan:
1. **Tema Bunga:** Sentuhan floral yang romantis dan natural.
2. **Tema Simple:** Minimalis, bersih, dan modern.
3. **Tema Elegant:** Fitur lengkap (Love Story, Galeri, Countdown).

## 🚀 Teknologi yang Digunakan

- **HTML5 & CSS3:** Struktur dan styling modern tanpa library eksternal (Pure CSS).
- **GitHub Pages:** Hosting gratis dan mudah untuk demonstrasi katalog.
- **gh-pages:** Package untuk otomatisasi deployment ke GitHub Pages.

## 🛠️ Pengembangan & Deployment

### Instalasi
Jika Anda ingin mengembangkan landing page ini secara lokal:
```bash
npm install
```

### Deployment
Proyek ini dikonfigurasi untuk dideploy ke GitHub Pages menggunakan perintah:
```bash
npm run deploy
```
